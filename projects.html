CNNs-for-breast-cancer-detection-and-diagnosis

The aim of this project is to develop CAD system by mean of CNNs to perform abnormality classification in mammographies. Both from scrach and ImageNet pretrained state-of-the-art CNNs were experimented. The entire project has been implemented by exploiting Keras Deep Learning framework.

Sketches-Web-Search-Engine

Multimedia Information Retrieval & Computer Vision Project The project consists in a web application in which a user can draw a sketch as input, and the system will retrieve and return the most similar sketches back to the user. The system is based on a CNN architecture using InceptionV3 pretrained on ImageNet as convolutional base for deep features extraction from sketches images. The deep features are indexed by mean of a LSH index.

Automated-OpenStack-Cloud-Infrastructure

A simple python script for the automation of a basic OpenStack based cloud infrastructure of Virtual Machines. The script automates the creation of new VMs at scheduled times and destroys them after a certain period. The application exposes a REST interface through which an hyphotetic system administrator can specify the VMs creation period beginning and end.
